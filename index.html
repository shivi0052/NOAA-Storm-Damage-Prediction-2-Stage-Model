The Core Challenge

73.7% of storms cause zero damage.

73.7%
26.3%
No damage recorded Damage recorded

A single regression model trained on this data learns to predict numbers close to zero for everything. The fix is to split the question in two, the same way professional catastrophe models work.

Stage 1, Classification

Will this storm cause any damage?

Trained on all 283,155 events using Logistic Regression, then Random Forest, then XGBoost. Output: a damage or no damage flag, the moment NOAA logs the event.

Stage 2, Regression

If damage occurs, how much?

Trained only on the 74,514 damage positive events using Linear Regression, then Random Forest, then XGBoost. Output: an estimated dollar figure for reserving.

What an insurer should actually do with this.

Built for a simulated regional insurance company. Every recommendation states its evidence and its honest limit.

01

Flag damaged locations the same day, not days later

The moment NOAA logs a storm's type, magnitude, and location, the model flags which locations likely suffered damage. This replaces a manual field adjuster process that takes days.

80.2% damage event recall, 96.5% of no damage events correctly cleared

See losses clustering before the claims flood in

Mapping flagged damage by location surfaces regional accumulation early, for example multiple counties hit by the same hurricane, giving finance and reinsurance teams advance warning.

03

Set financial reserves the same day

For every flagged event, the model also produces a dollar estimate, giving finance a working number immediately instead of waiting weeks for field reports.

R squared of 0.653, estimates typically within about 3.4 times the actual dollar figure
04

Add exposure data for full catastrophe model capability

The current model is hazard only. Joining it with property exposure data such as building value, age, and construction type is the clear next phase.

Hazard only R squared of 0.65 is consistent with literature. Exposure data can push this toward 0.80 to 0.90
Ethical Considerations

Geography is a hazard signal, not a proxy.

This model uses geographic features such as state, coastal indicator, and latitude and longitude as predictors. Storm risk genuinely varies by geography. Coastal exposure and hurricane paths are physical phenomena, not stand ins for protected demographic characteristics.

That said, any operational deployment should ensure damage estimates and reserve recommendations do not inadvertently slow claims response in lower income or historically underserved regions, since disaster recovery speed already correlates with such factors independent of this model.
